# Projeto Casamento — Lucas & Amanda

Site para celebração do Sacramento do Matrimônio de Lucas e Amanda.

## Funcionalidades
- Carrossel de Fotos
- Contador Regressivo
- Buquê Espiritual (Corrente de Oração)

## Tecnologias
- React + Vite
- Tailwind CSS
- Prisma (PostgreSQL)

## Desenvolvimento
```sh
npm install
npm run dev
```
