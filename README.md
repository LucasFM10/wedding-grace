# Projeto Casamento — Lucas & Amanda

Site para celebração do Sacramento do Matrimônio de Lucas e Amanda.

## Funcionalidades

- Carrossel de Fotos
- Contador Regressivo
- ElevadorParaDois (Corrente de Oração)

## Tecnologias

- Next.js
- React
- Tailwind CSS
- Supabase (PostgreSQL)
- shadcn/ui

## Desenvolvimento

```sh
npm install
npm run dev
```
